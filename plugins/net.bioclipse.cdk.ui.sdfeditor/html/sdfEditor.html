The SDF Editor

This editor displays entries in SD files and multiple molecule CML files and properties in the file. In the Bioclipse Navigator these files show the icon and an arrow pointing on the file can be used to expand its children. Double clicking the file opens it in the editor.

The editor shows the structures and the properties on the Molecules tab. The Headers tab makes it possible to choose which properties (those in the "Visible properties" column) of all properties ("Available properties") are shown on the Molecules tab. Use Add and Remove buttons to add and remove properties from the Visible selection.
